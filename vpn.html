科学上网教程

本文迁移自华南虎科学上网教程，并已对相关链接作相应修改。

Windows

Windows下常用的代理软件有v2rayN、Clash、Qv2ray、Netch等。本人推荐使用Clash，图形化界面在Windows的可使用性比较强。

重要提示：如果你没有关闭代理功能就直接重启或关闭电脑，开机后访问网页将会出现代理服务器有误的报错，此时打开代理软件再退出即可恢复正常。

v2rayN教程

第一步：安装v2rayN

准备v2rayN的安装包，文件为zip压缩包

下载地址：Github Release(官方，可能需要VPN)、华南虎资源站（校园网）、百度网盘

第二步：配置客户端

解压后进入文件夹，打开 v2rayN.exe

点击左上方订阅->订阅设置，添加一个订阅，备注填入 (自己命名)，地址填入 https://api.sub-100.one/link/6iWOKtRVCKcoyzPV?clash=3&extend=1，勾选启用选项，点击确定按钮

点击左上方订阅->更新订阅，软件自动下载配置文件

更新完成后将出现我们科学上网服务的节点信息，选择需要的节点，点击回车即可设为活动节点

在软件任务栏图标上右键，选择系统代理->自动配置系统代理，软件图标变红即开启成功

至此，v2rayN配置完成，在浏览器中输入 https://google.com 测试是否能访问谷歌搜索，如果成功加载则配置成功。

Clash教程

Clash目前缺少对VLESS与XTLS的支持，部分线路将无法使用

第一步：安装Clash

准备Clash的安装包，根据系统位数下载对应exe文件（ia32为32位安装包）

下载地址：Github Release、华南虎资源站、百度网盘

第二步：配置客户端

安装完成以后打开，主页设置可按如下修改，倒数第二项的 System Proxy 先别打开

选择左侧的Profiles选项，点击右边的 ×，删除自带的测试文件 config.yaml

在上方文本框填入 https://api.sub-100.one/link/6iWOKtRVCKcoyzPV?clash=3&extend=1，然后点击右边的Download按钮下载配置文件

点击右边的设置按钮修改配置名称，自动更新时间按需设置

选择左侧的Proxies选项，上方选择Rule模式并选择节点，同时按需调整各服务分流配置

回到General选项，打开System Proxy开关

至此，Clash配置完成，在浏览器中输入 https://google.com 测试是否能访问谷歌搜索，如果成功加载则配置成功。

Android

v2rayNG教程

第一步：安装v2rayNG

准备v2rayNG的APK安装包，选择对应设备SOC架构的版本（正常手机选择arm64-v8a版本即可）

下载地址：Github Release、华南虎资源站、百度网盘

如果你的手机带有Play Store，同时也能正常访问谷歌服务，也可以在这里直接安装，注意谷歌商店的版本会更新滞后。

第二步：配置客户端

安装完成后打开，点击左上角打开侧面板->订阅设置

点击右上角的加号,填入配置信息，备注填入 自己命名，地址填入 https://api.sub-100.one/link/6iWOKtRVCKcoyzPV?clash=3&extend=1，勾选启用选项，点击右上角确认

回到主页，点击右上角的更新订阅

选择合适的节点，点击右下角的V按钮开启服务

至此，v2rayNG配置完成，在浏览器中输入 https://google.com 测试是否能访问谷歌搜索，如果成功加载则配置成功。

Shadowrocket教程

第一步：安装Shadowrocket

由于App Store中国大陆区的Shadowrocket（小火箭）已经被下架，目前只能在外区才能下载到这个app，一般情况下会选择美区账号下载。

注意，如果你在国区App Store直接搜索Shadowrocket，将会有一个同名同图标的钓鱼软件，切勿下载使用

我们的目标就是安装上一个能用的小火箭应用，其基础就是获取到一个购买过小火箭的Apple账号，一般有如下方法

  1. 淘宝或闲鱼，你在上面可以找到这种共享账号，按照它提供的方式下载安装即可，缺点是更新时需要重新登录对方的ID才能下载。

  2. 自己注册美区ID并购买，你需要先退出本机Apple ID，修改本机语言为英文，本机位置为美国，正常情况下建议挂上美国的梯子。在登录Apple ID的界面选择注册新的iCloud邮箱，按说明操作即可得到一个美区ID，接着打开APP Store，搜索Shadowrocket，安装时会要求你填写付款资料，如果你有国外信用卡（国内visa或者万事达卡不可用）或者美区Paypal的话可以按情况填写，如果没有的话这里选择跳过，然后到淘宝购买美区APP Store兑换码，充值以后即可购买。

    3. 此处提供一个美区共享账号，由梯子提供方提供，切勿长时间登录此账号，特别地不能使用该账号登录icloud！！，以免导致手机被锁，账户名：l-efy4j.4@protonmail.com 密码：HDrj044932

第二步：配置小火箭

打开Shadowrocket

点击软件右上角的加号，类型修改为Subscribe，URL填入https://api.sub-100.one/link/6iWOKtRVCKcoyzPV?shadowrocket=1&extend=1，而后点击右上角完成按钮。

Linux

Qv2ray教程

第一步：安装Qv2ray

准备Qv2ray的安装包，文件为AppImage格式，同时下载Xray内核文件

下载地址：Github Release(Qv2ray)、Github Release(Xray_core)、华南虎资源站、百度网盘

这里以Ubuntu举例，各发行版操作大致相同，不过部分桌面环境可能无法自动开启系统代理，需要在浏览器手动配置，目前测试GNOME与KDE均可正常工作。

第二步：配置客户端

右键Qv2ray程序，选择属性，勾选 Allow executing file as program ，或者使用 chmod 为该文件加上运行属性。

双击Qv2ray程序，可打开主界面

打开命令行，解压Xray内核到 ~/.config/qv2ray/xray-core/ 文件夹下

# 如果系统未安装zip与unzip功能，请先执行以下命令
shell> sudo apt install zip unzip
···
shell> unzip -d ~/.config/qv2ray/xray-core Xray-linux-64.zip
···

点击左上方Preferences->Kernel Settings，内核路径修改为 /home/dnomd343/.config/qv2ray/xray-core/xray，资源路径修改为 /home/dnomd343/.config/qv2ray/xray-core/，注意用户名改为自己账户名，然后点击 Check V2ray Core Settings 检查内核是否正常，如果与图中类似即安装成功，再点击右下角OK按钮退出设置界面

点击左上方Groups->Subscription Settings，填入组名为 （个人喜好命名），勾选 This group is a subscription，地址填入 https://api.sub-100.one/link/6iWOKtRVCKcoyzPV?clash=3&extend=1 ，点击下方 Update Subscription，在确认窗口点击Yes更新订阅

进入 Connections 选项卡可看到订阅的线路信息

回到主页，双击 你的命名 展开该组，选择需要的线路双击即可选中并连接

至此，Qv2ray配置完成，在浏览器中输入 https://google.com 测试是否能访问谷歌搜索，如果成功加载则配置成功。
